W3Cliability,trademark,documentuse andsoftwarelicensing rules apply.AbstractThis document provides techniques for satisfying the checkpoints defined in"User Agent Accessibility Guidelines 1.0"title="Link to reference UAAG10">[UAAG10]. These techniques addresskey aspects of the accessibility of user interfaces, content rendering,application programming interfaces (APIs), and languagessuch as the Hypertext Markup Language(HTML), Cascading StyleSheets (CSS) and theSynchronized Multimedia Integration Language(SMIL).The techniques listed in this document are not required for conformance tothe Guidelines. These techniques are not necessarily the only way of satisfyingthe checkpoint, nor are they a definitive set of requirements for satisfying acheckpoint.Status of this documentThis section describes the status of this document at the time of itspublication.
